О проекте

W.A.Y. помогает выбрать направление без давления.

Платформа соединяет карьерный тест, каталог профессий и AI-наставника в один спокойный маршрут. Сначала структура и данные, потом аккуратная интерпретация.

Для учеников

Понятный язык, короткие шаги и профессии из реального каталога.

Для родителей

Результаты объясняют, почему направление подходит, а не просто называют профессию.

Для школы

Админ-страница показывает пользователей, активность и завершенные тесты.
